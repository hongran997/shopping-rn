### React 与 Vue 的不同
1. React 会通过props 把children 传递给另一个组件
2. React 函数的概念很强【不好设置参数来查看效果】 Vue组件的概念很强【很方便查看一个组件的效果】
3. JSX 的语法逻辑使得代码不好读，也不好写，看着也不整齐，尤其是遇到多个if, else的时候，用很多的？： 简直要崩溃
4. 好烦啊，需要跳着组件看。
5. 偶尔看下下面的warning 也是有用的啊。
6. 还是从小组件开始开发，从内部组件开始开发
7. 函数的方式难组织，但灵活性更强了
